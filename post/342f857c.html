    01、什么时Promise对像

    Promise是ES6中一个非常重要和好用的特性，是异步编程的一种解决方案。

    02、使用

    2.1、基本使用

          const p = new Promise(function(resolve, reject) {
        let data=... //网络请求
      });
      p.then(function(data) {
        console.log(data)
      })

    promise对像有两个固定参数：resolve和resject.

    如果请求成功，调用resolve。接着就会调用then

    如果请求失败，调用reject。接着就会调用 catch

    2.2、链式调用

          new Promise((resolve, reject) => {
         let data=... //网络请求
      }).then(data => {
        console.log(data)
        return Promise.resolve(data + '111')
      }).then(data => {
        console.log(data)
        return Promise.resolve(data + '222')
      }).then(data => {
        console.log(data)
      })

    then和catch都返回了一个Promise，所以可以链式调用

    Promise.resolve()：将数据包装成Promise对像，并且在内部回调 resolve

    Promise.reject()：将数据包装Promise对像，并且在内部回调reject

    注意：如果我们直接返回一个数据，那么默认就是调用了Promise.resolve
