    01、 插值操作

    1.1、插值运算符

    语法：

    <h2>{{name}}</h2>

    1.2、v-once、

    该指令表示：元素和组件之间只会渲染一次，页面不会随着数据的改变而改变。

    <h2 v-once>{{firstName}} {{lastName}}</h2>

    1.3、v-html

    <!--如果数据本身就是html代码，就不能用{{}}输出，这时候就用v-html进行格式输出-->
<h2 v-html="htmlData"></h2>

    1.4、v-text

    v-text作用和插值运算符类似，不同的是v-text会替换标签内的内容

    <h2 v-text="name">Hello </h2>

    1.5、v-pre

    不编译，原样输出。如下，会输出：{{firstName}}

    <h2 v-pre>{{firstName}}</h2>

    1.6、v-clock

    防止浏览器直接显示未编译的插值运算符

    <h2 v-clock>{{firstName}}</h2>
<style>
    [v-clock] {
      display: none;
    }
</style>

    02、filter 过滤器

    如果得到的数据不符合我们的需求，就需要将其改变，或使用默认值。filter与data和methods平级。

    <body>
  <div id="app">
    <h2>{{num | filterEven}}</h2>
  </div>
</body>
<script>
  let app = new Vue({
    el: '#app',
    data: {
      num: 10,
    },
    filters: {
      filterEven(num) {
        if (num % 2 === 0) {
          return num
        } else {
          return '非法数字'
        }
      }
    }
  })
</script>

    03、v-bind 属性绑定

    3.1、绑定一个属性

    <body>
  <div id="app">
    <a v-bind:href="baiduLink">百度</a>
    <!-- v-bind有简写方式 -->
    <a :href="baiduLink">百度</a>
  </div>
</body>
<script>
  let app = new Vue({
    el: '#app',
    data: {
      baiduLink: 'http://www.baidu.com'
    }
  })
</script>

    3.2、绑定class

    <body>
  <div id="app">
    <!-- 方式1：直接通过{}绑定一个类 -->
    <h2 :class="{'active':isActive}">Hello Vue</h2>
    <!-- 方式2：这种方式可以通过判断，传入多个值 -->
    <h2 :class="{'active':isActive, 'size': big}">Hello Vue</h2>
    <!-- 方式3：和普通类同时存在，并不冲突 -->
    <h2 class="h2-border" :class="{'active':isActive, 'size': big}">Hello Vue</h2>
    <!-- 方式4：如果过于复杂，可以放到一个data中 -->
    <h2 class="h2-border" :class="classes">Hello Vue</h2>
  </div>
</body>
<script>
  let app = new Vue({
    el: '#app',
    data: {
      isActive: true,
      big: true,
      classes: ['active', 'size']
    }
  })
</script>

    3.3、绑定style

    <body>
  <div id="app">
    <!-- 方式1：直接绑定对象 -->
    <h2 :style="{color: 'red', fontSize: '15px'}">Hello Vue</h2>
       <!-- 方式2：绑定style -->
     <h2 :style="style1">Oye Como Va</h2>
  </div>
</body>
<script>
 let app = new Vue({
    el: '#app',
    data: {
         style1: {color: 'red', fontSize: '15px'}//方式2的数据

    }
  })
</script>

    04、监听器

    <body>
  <div id="app">
    <input type="text" v-model="firstName">
    <input type="text" v-model="lastName">
    <h2>{{fullName}}</h2>
  </div>
</body>
<script>
  let app = new Vue({
    el: '#app',
    data: {
      firstName: '张',
      lastName: '狗子',
      fullName: ''
    },
    watch: {
      firstName: function(newVal, oldVal) {
        // 在vue中，操作data、method必须使用this
        this.fullName = this.firstName + this.lastName
      },
      lastName(newVal, oldVal) {
        this.fullName = this.firstName + this.lastName
      }
    }
  })
</script>

    05、计算属性

    5.1、使用

    <body>
  <div id="app">
    <input type="text" v-model="firstName">
    <input type="text" v-model="lastName">
    <h2>{{fullName}}</h2>
  </div>
</body>
<script>
  let app = new Vue({
    el: '#app',
    data: {
      firstName: '张',
      lastName: '狗子'
    },
    computed: {
      fullName() {
        // 写法是个方法，但是使用的时候是作为属性使用的，和data一致。
        //只要data中的数据发生变化，就会重新计算，不会使用缓存中的值
        return this.firstName + this.lastName
      }
    }
  })
</script>

    计算属性和监听器的区别：

    computed：以方法形式写，以属性形式使用。并且结果会被缓存。

    methods：表示方法，主要写业务逻辑

    watch：是一个对象，key是需要监听的表达式，value是回调函数。主要用来监听数据变化 。

    5.2、计算属性的getter和setter方法

    想要改变计算属性的值，就使用get()和set()方法

    <body>
  <div id="app">
    <input type="text" v-model="firstName">
    <input type="text" v-model="lastName">
    <input type="text" v-model="fullName">
  </div>
</body>
<script>
  let app = new Vue({
    el: '#app',
    data: {
      firstName: '张',
      lastName: '狗子'
    },
    computed: {
      fullName: {
        get() {
          console.log('调用了get方法')
          return this.firstName+' '+this.lastName
        },
        set(val) {
          console.log('调用了set方法')
          const names = val.split(' ')
          this.firstName = names[0]
          this.lastName = names[1]
        }
      }
    }
  })
</script>

    计算属性的结果会被缓存起来，如果计算属性使用的数据来自data，那么只要data的数据发生变化，就会重新计算、重新求值。

    计算属性中一定有一个return回来的值，用来代表这个属性。

    06、v-on（@）事件监听

    <body>
  <div id="app">
    <h2>{{count}}</h2>
    <input type="text" v-on:focus="alertMsg">
    <button v-on:click="addCount">点我</button>
    <!-- v-on的简写方式 -->
    <button @click="addCount">点我</button>
  </div>
</body>
<script>
  let app = new Vue({
    el: '#app',
    data: {
      count: 0
    },
    methods: {
      addCount() {//如果需要传参数，就在括号中加个参数即可
        this.count++
      },
      alertMsg() {
        alert('获取了焦点')
      }
    }
  })
</script>

    07、条件判断

    7.1、v-if、v-else-if、v-else

    <body>
  <div id="app">
    <div v-if="score >= 90">优秀</div>
    <div v-else-if="score >= 75">良好</div>
    <div v-else-if="score >= 60">及格</div>
    <div v-else>不及格</div>
    <button @click="delScore">减分</button>
  </div>
</body>
<script>
  let app = new Vue({
    el: '#app',
    data: {
      score: 100
    },
    methods: {
      delScore() {
        this.score -= 10
      }
    }
  })
</script>

    7.2、v-show

    <body>
  <div id="app">
    <div v-show="score >= 90">优秀</div>
    <div v-show="score >= 75 && score < 90">良好</div>
    <div v-show="score >= 60 && score < 75">及格</div>
    <div v-show="score < 60">不及格</div>
    <button @click="delScore">减分</button>
  </div>
</body>
<script>
  let app = new Vue({
    el: '#app',
    data: {
      score: 100
    },
    methods: {
      delScore() {
        this.score -= 10
      }
    }
  })
</script>

    v-if与v-show的区别：

    v-if：不符合条件就不创建DOM元素，

    v-show：不符合条件也创建DOM元素，只是使用displaya;none隐藏了该元素。降低了效率。
