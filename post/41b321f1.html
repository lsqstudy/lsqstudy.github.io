    02、插件

    2.1常用插件

    1. HTML Snippets： 超级实用且初级的 H5代码片段以及提示
    2. HTMLHint： html代码检测
    3. HTML CSS Support : 让 html 标签上写class 智能提示当前项目所支持的样式。新版已经支持scss文件检索，这个也是必备插件之一
    4. Path Autocomplete : 路径智能补全
    5. Path Intellisense ： 路径智能提示
    6. JavaScript Snippet Pack： 针对js的插件，包含了js的常用语法关键字，很实用；
    7. View InBrowser： 从浏览器中查看html文件，使用系统的当前默认浏览器
    8. Class autocomplete for HTML： 编写html代码的朋友们对html代码的一大体现就是重复，如果纯用手敲不仅累还会影响项目进度，这款自动补全插件真的很棒；
    9. beautify : 格式化代码的工具，可以格式化JSON|JS|HTML|CSS|SCSS,比内置格式化好用
    10. Debugger for Chrome： 让 vscode 映射 chrome 的 debug功能，静态页面都可以用 vscode 来打断点调试，真666~
    11. jQuery Code Snippets： jquery 重度患者必须品
    12. vscode-icon： 让 vscode 资源树目录加上图标，必备良品！
    13. VSCode Great Icons： 另一款资源树目录图标
    14. Code Runner : 代码编译运行看结果，支持众多语言
    15. GitLens： 丰富的git日志插件
    16. vetur： vue语法高亮、智能感知、Emmet等
    17. VueHelper： vue代码提示
    18. open in browser： html等等页面可以右键打开
    19. Live Server： 开启一个服务，好像tomcat，默认端口：5500
